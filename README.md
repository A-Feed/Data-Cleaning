# Data-Cleaning
Terdapat 2 dataset mentah yang diambil dari kaggle tentang pola hidup ikan air tawar.
Kaggle: https://www.kaggle.com/datasets/monirmukul/realtime-pond-water-dataset-for-fish-farming
        https://www.kaggle.com/datasets/muzamilaslam/fish-life-prediction
        
Data tersebut diolah menjadi 3 dataset:
* Data_clean (2 data yang digabungkan kemudian sudah dibersihkan)
* Data_type_ikan (dataset yang memiliki type ikan yang dibutuhkan)
* Data_label_optimal(dataset yang memiliki label optimal dan kurang optimal)

Pada data_label_optimal nantinya akan diolah lebih lanjut untuk pembuatan model.
